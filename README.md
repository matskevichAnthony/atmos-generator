# atmos-generator

Набор браузерных генераторов индастриал-техно звука на движке
[Strudel](https://strudel.cc) (`@strudel/web` / superdough). Без сборки и
фреймворков: чистый ES-модульный vanilla JS, статика раздаётся крошечным
Node-сервером. Всё, что генерится, можно записать в **WAV** и утащить в DAW.

## Быстрый старт

```bash
npm install        # ставит @strudel/web и @strudel/repl
npm start          # поднимает http://localhost:5173
```

Открой `http://localhost:5173` — на главной есть навигация по всем приборам.
Звук в браузере включается только после клика (autoplay policy), поэтому в
каждом приборе жми **PLAY**.

## Приборы

| URL | Что делает |
|-----|------------|
| `/` | Живой REPL со стартовым индастриал-лупом (`<strudel-editor>`) + REC WAV |
| `/kick/` | Синтез-кик: питч-огибающая, дисторшн; пресеты Techno / Gabber / Rumble |
| `/rumble/` | Рамбловый индастриал-кик, 4 способа генерации хвоста: reverb / decay / delay / sub |
| `/atmos/` | Долгие триповые атмосферы: аккорд-дрон + медленная модуляция фильтра/вибрато/фейзера |
| `/studio/` | Многослойная студия трека: микшер слоёв (кик, рамбл, хэты, перк, эйсид, атмо, эйр), запись стемов и микса по тактам |
| `/corruptor/` | **DC-77 DATA CORRUPTOR** — сидовый генератор странных FX/атмосфер с двумя стойками порчи и картинко-в-звук. Флагман, см. ниже |

## Запись в WAV

`shared/capture.js` подключён на каждой странице первым скриптом в `<head>`.
Он патчит конструктор `AudioContext` **до** старта движка, вешает «отвод» на
мастер-выход и кодирует сырой PCM в **16-bit WAV** (Ableton/любая DAW открывают
перетаскиванием; браузерный webm/opus не годится). Есть анти-клип нормализация.

## DC-77 DATA CORRUPTOR (`/corruptor/`)

Самый глубокий прибор. Философия: **самодостаточный генератор коллапсирующего
звука**, всё детерминировано от 8-значного HEX-сида.

- **SOURCE** — SHOT (удар) / LOOP (ритм-петля) / DRONE (тянущийся тон), регистр,
  число нот, `⌁ NOTES` перебрасывает мелодию (порядок+высоты), не трогая ритм.
- **RACK A · LIVE** — 15 модулей эффектов поверх живого стрима Strudel (мгновенно).
- **RACK B · COLLAPSE ENGINE** — 8 офлайн-модулей порчи над отрендеренными
  сэмплами (флипы битов, застрявший буфер, потеря секторов, спектральные
  robot/smear через свой FFT). Кривая CURVE распределяет порчу по длине файла.
- **IMAGE UNIT** — картинка → звук: SPECTR (как спектрограмма), CARVE
  (выжигается в спектр), BEND (байты файла как PCM). Есть встроенный
  процедурный генератор картинок `GEN IMG`.

Один **PLAY** сам выбирает: живой стрим (source+RackA) или зацикленный
офлайн-рендер (как только включён RACK B / картинка). Длина файла задаётся в
секундах, **REC WAV** пишет ровно эту длину. Состояние сессии переживает
перезагрузку (localStorage), звук воспроизводим по сиду.

Подробная карта модулей и внутренние механики — в [`ARCHITECTURE.md`](./ARCHITECTURE.md).

## Стек и конвенции

- **Vanilla JS, ES-модули, zero-build.** Движок грузится как самодостаточный
  бандл: `import … from '/node_modules/@strudel/web/dist/index.mjs'`.
- **RSJS-стиль**: хуки поведения через `data-js-*` (в JS это camelCase:
  `data-js-preset` → `el.dataset.jsPreset`), а `data-*` — опции/данные. Классы
  только для стилей.
- **Разделение по файлам**: разметка / стили / поведение раздельно; логика
  разложена по модулям (особенно в `/corruptor/`).
- **Сервер** (`server.mjs`) — zero-dependency статика, корень = каталог самого
  файла, есть MIME-карта и directory-index.

## Оформление

Брутализм-терминал: чистый чёрный / белый / один горячий красный `#ff0000`,
нулевые скругления, жёсткие 1px-рамки, пиксельный шрифт **VT323**, сканлайны.
